WhiteLion

Download

WhiteLion is a theme for Growl developed by @caio_gondim that tries to be as Lion-esque as possible. Some UI of the new OS X, like the new QuickLook and Popovers, served as the basis for this theme.

Built to Beauty

This theme was very carefully pixel-perfect built. If you have discovered some errors or have some considerations to make, please, open an issue here.

Open Source

This project is open source. Please, feel free to fork it or open an issue to help make this better for everybody.

Inspiration

WhiteLion is heavily inspired in Mono by Christopher Lobay and GrowlingLion by Andreas Heiberg
